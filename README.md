# Grauwoelfchen's Canvas

The website of https://grauwoelfchen.at/


## Setup

```zsh
% python -V
3.5.4
% python -m venv venv
% source ./venv/bin/activate
(venv) % pip install --upgrade pip setuptools
(venv) % pip install Nikola -c constraitns.txt
```


## Serve

```zsh
(venv) % cd site
(venv) % nikola serve
```


## Build

```zsh
% nikola build
```
