# Grauwoelfchen's Canvas

The website of https://grauwoelfchen.at/


## Setup

```zsh
% rustc --version
rustc 1.43.1 (8d69840ab 2020-05-04)

# install nib-cli and nib-server
% make setup
```


## Serve

```zsh
% make serve
```


## Build

```zsh
% make build
```
